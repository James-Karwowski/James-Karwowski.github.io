Pong 2: 3DS Edition

Pong 2: 3DS Edition is a homebrew game I developed for the Nintendo 3DS using C++ and the DevKitPro toolchain. The game was created as part of a personal project to learn more about game development and homebrew programming for handheld consoles as well as to give me a challenge.

Secure Password Generator

The Password Generator is a simple web application that allows users to create random passwords based on their specified criteria. Users can choose the length of the password and whether to include uppercase letters, lowercase letters, numbers, and special characters. The application is built using HTML, CSS, and JavaScript.
